# Contributing

## Local workflow

1. Use the project Node.js version:

```bash
nvm use
```

2. Install dependencies:

```bash
yarn install
```

3. Run the TypeScript check:

```bash
yarn check
```

## Pull requests

- Keep configuration examples sanitized.
- Do not commit `credentials.json`, `.env`, or Gmail tokens.
- Update `README.md` if setup or deployment behavior changes.
